This repository contains hands-on AWS Cloud Practitioner–level projects demonstrating core AWS services through real-time practical implementations. The goal of this repository is to build strong cloud fundamentals and showcase practical AWS experience for entry-level cloud roles.

AWS Services Covered :

Amazon EC2 – Instance launch, security groups, monitoring Amazon S3 – Bucket creation, versioning, event notifications AWS IAM – Users, roles, policies, security best practices Amazon CloudWatch – Metrics, alarms, SNS notifications Amazon SQS – Standard queues, producer-consumer model
